# Timeframes | Enumeration
Timeframes enumeration identifiers are used in `CTicks->`[Select()](../modules/main/ticks.md).

### ENUM_NAME
| ID        | Description           | Type |
| :-------- | :-------------------- | :--- |
| `CURRENT` | The current timeframe | int  |
| `M1`      | 1 minute              | int  |
| `M5`      | 5 minutes             | int  |
| `M15`     | 15 minutes            | int  |
| `M30`     | 30 minutes            | int  |
| `H1`      | 1 hour                | int  |
| `H4`      | 4 hours               | int  |
| `D1`      | 1 day                 | int  |
| `W1`      | 1 week                | int  |
| `MN1`     | 1 month               | int  |
